# PDF 裁剪功能

## 概述

可视化PDF裁剪工具，支持实时预览和自由选择裁剪区域。

## 功能特性

### 1. PDF预览
- 实时渲染PDF页面
- 支持翻页浏览
- 清晰的页面显示

### 2. 可视化裁剪
- **拖拽移动**：点击蓝色区域拖动裁剪框
- **调整大小**：拖拽四个角的控制点调整裁剪框大小
- **实时反馈**：蓝色区域为保留内容，灰色区域为裁剪部分

### 3. 应用范围
- **仅当前页**：只裁剪当前预览的页面
- **所有页面**：对PDF的所有页面应用相同的裁剪
- **指定范围**：支持灵活的页面范围语法
  - 单页：`1, 3, 5`
  - 范围：`1-5`
  - 混合：`1-5, 8, 10-12`

### 4. 本地处理
- 使用 pdf-lib 和 pdf.js 在浏览器中处理
- 无需上传到服务器
- 保护用户隐私

## 使用流程

1. **上传文件**
   - 拖拽PDF文件到上传区域
   - 或点击"点击上传文件"按钮选择文件
   - 文件大小限制：100MB

2. **选择裁剪区域**
   - 在PDF预览上看到蓝色裁剪框
   - 拖拽蓝色区域移动裁剪框
   - 拖拽四个角调整裁剪框大小
   - 蓝色区域为保留的内容

3. **选择应用范围**
   - 默认应用到所有页面
   - 可选择仅当前页或指定页面范围

4. **应用裁剪**
   - 点击"应用裁剪"按钮
   - 等待处理完成（显示进度条）

5. **下载结果**
   - 处理完成后点击"下载文件"
   - 文件名格式：`原文件名_cropped.pdf`

## 技术实现

### 核心依赖
- `pdf-lib`: PDF操作库
- `pdfjs-dist`: PDF渲染库
- `@element-plus`: UI组件
- `@iconify/vue`: 图标

### 关键功能

#### PDF渲染
```typescript
const page = await pdfDoc.getPage(pageNum)
const viewport = page.getViewport({ scale: 1.5 })
await page.render({ canvasContext: context, viewport }).promise
```

#### 裁剪框交互
- 拖拽移动：监听 mousedown/mousemove/mouseup 事件
- 调整大小：四个角的控制点独立处理
- 边界限制：确保裁剪框不超出页面范围

#### 坐标转换
```typescript
// 像素坐标 → PDF坐标
const cropX = cropBox.x / scale
const cropY = cropBox.y / scale

// PDF坐标系转换（左下角为原点）
const pdfY = height - cropY - cropHeight
```

## 操作技巧

### 精确裁剪
1. 先大致拖拽到目标位置
2. 使用角控制点精确调整大小
3. 微调位置确保内容完整

### 批量处理
1. 在第一页设置好裁剪区域
2. 选择"所有页面"
3. 一次性处理所有页面

### 部分页面裁剪
1. 设置裁剪区域
2. 选择"指定范围"
3. 输入需要裁剪的页面范围

## 常见问题

### Q: 裁剪框太小怎么办？
A: 拖拽四个角的控制点可以放大裁剪框。

### Q: 如何重置裁剪框？
A: 点击"重置"按钮，裁剪框会恢复到整个页面大小。

### Q: 不同页面可以设置不同裁剪区域吗？
A: 当前版本所有选中页面使用相同的裁剪区域。

### Q: 裁剪后文件变大了？
A: 这是正常的，pdf-lib会重新保存PDF。可以使用"PDF压缩"工具进一步压缩。
