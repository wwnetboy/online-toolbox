# 图片去水印模型使用说明

## 模型文件

- **文件名**: `migan_pipeline_v2.onnx`
- **大小**: 约 28MB
- **位置**: `/public/models/migan_pipeline_v2.onnx`
- **用途**: 图片去水印 AI 模型

## ONNX Runtime WASM 文件

图片去水印功能需要 ONNX Runtime 的 WASM 运行时文件。

### 同步 WASM 文件

首次使用或更新 `onnxruntime-web` 依赖后，需要运行：

```bash
pnpm sync:onnxruntime-wasm
```

或者：

```bash
node scripts/sync-onnxruntime-wasm.cjs
```

这会将以下文件从 `node_modules/onnxruntime-web/dist` 复制到 `public/onnx/`：

- ort-wasm-simd-threaded.mjs
- ort-wasm-simd-threaded.jsep.mjs
- ort-wasm-simd-threaded.asyncify.mjs
- ort-wasm-simd-threaded.wasm
- ort-wasm-simd-threaded.jsep.wasm
- ort-wasm-simd-threaded.asyncify.wasm

## 工作原理

1. **本地加载**: 所有模型和 WASM 文件都从本地加载，不需要联网
2. **首次加载**: 第一次使用时会加载模型到内存（约需几秒）
3. **缓存复用**: 模型加载后会缓存在内存中，后续使用无需重新加载
4. **离线运行**: 所有 AI 处理都在浏览器本地完成，保护用户隐私

## 故障排除

### 错误：模型加载失败

**原因**: WASM 运行时文件缺失

**解决方案**:
```bash
cd Toolbox
pnpm sync:onnxruntime-wasm
```

### 错误：本地模型不存在

**原因**: 模型文件缺失或路径错误

**解决方案**:
1. 确认 `public/models/migan_pipeline_v2.onnx` 文件存在
2. 确认开发服务器正在运行
3. 检查浏览器控制台的详细错误信息

## 技术细节

- **模型**: MI-GAN (Mask-Inpainting GAN)
- **运行时**: ONNX Runtime Web (WASM)
- **执行提供者**: WASM (不使用 WebGPU，因为模型需要 uint8 输入)
- **多线程**: 支持（使用 SharedArrayBuffer）
- **SIMD**: 支持（加速计算）
